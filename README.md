shadowsocks-heroku-winxp
==================

shadowsocks-heroku-winxp is a client for shadowsocks-heoku (https://github.com/mrluanma/shadowsocks-heroku), which is compatible with node < v6, so it can be used in win xp that is not supported since version 6.

for the server use shadowsocks-heoku (https://github.com/mrluanma/shadowsocks-heroku).
